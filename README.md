# Population-in-CZ
 ETL process + visualization for population in Czech Republic.

# Goal
- Display and comparison of historical population development in Prague and the South Moravian Region.

# Technical Assignment
- Download data from a REST API source.
- Create a simple dimensional model in a database.
- Create a simple chart in a reporting tool.

# Data Source
- REST API: https://api.store/czechia-api/czso.cz/dokumentace
	- Endpoint: https://api.store/czechia-api/czso.cz/dokumentace#tag/Obyvatelstvo-a-domy-v-obcich
	- Testing API Key - https://share.1password.com/s#akBP0ypn4SK2qWEA769LfbBotJTsYdpxa3YWdgOGvHU

# Steps to do before running the script
- Input the API key among the environmental variables.
- For connecting Power BI to SQLite3 database, you need to follow this article: https://community.fabric.microsoft.com/t5/Power-Query/connecting-a-SQLite-database/td-p/224205
